참여자 : 정성은, 김신현, 장형민
지각: 최자윤
사유: 인터넷이 안됨

서기: 정성은


주제: 프로토타입 방향성 설정

1. 현재까지 프로그래밍 진행도에 대한 논의

// 현재 완료라고 하는것은 기본적인 구조 틀이 완료되었다 라는것으로 상정하고 진행.

    1. 영웅 관련

        완료
        1. 캐릭터 데이터를 통해서 불러올 수 있는가.
        2. 캐릭터 스텟이 인게임 내 적용되며, 데이터를 받아 올 수 있는지.
        3. 캐릭터 스텟 변경이 적용 되는지 (외부적인 요인(특성, 아이템 등))

        미완료
        1. 아이템을 착용 할 수 있는지.
        2. 캐릭터의 이름이 제대로 출력되고 있는지.
        3. 훈련 과정 정보의 대한 테이블 작업 및 구조 작업이 필요함.

        불확실한 정보 (정리하여 다시 논의)
        1. 기본적인 특성이 완료 되었는가. (테이블 참조)
    
    2. 전투 관련
        
        완료
        1. 전투관련 기초적인 AI 구현
        2. 전투 내 로그가 제대로 출력되고 있는지
        3. 상태이상 일부 구현 상태로 프로토 마무리.

        미완료
        1. 전투 진행 중 오래 지속되는 전투를 종료하는 방법이 필요.
        2. 전투관련 스텟이 제대로 적용이 되었는지.(서브 스텟에 대한 내용이 구현 안됨)
        3. 상태이상이 다음 스테이지로 이전되는지.

    3. 스테이지 관련

        완료
        1. 다음 스테이지로 이동이 가능한지.
        2. 보스 스테이지가 제대로 표기 되는지.
        3. 중간 이벤트 출력이 되는지.(등장 확률에 대해서는, 하드 코딩 되어있음/ 개선 내용 참조 4번)

        미완료
        1. 스테이지 이동 시 이벤트 여부 확인이 가능한지.
        2. 스테이지 중간 휴식이 구현되어 있는지.
        3. 스테이지 종료(보스 처치 or 파티 전멸) 후 결과 화면이 있는지.

    4. 이벤트 관련

        완료
        1. 이벤트가 데이터를 통해서 출력되는지.

        미완료
        1. 이벤트의 추가 커맨드가 생길 경우 코드작업 해야됨

        불확실한 정보
        1. 이벤트 발생 확률이 하드 코딩 되어있음 
        >> 확률에 대한 하드 코딩은 스테이지 테이블 event_per 파라미터로 관리, 등장하는 이벤트의 확률의 경우 가챠 테이블을 통해 관리 가능하게 진행

2. 아직 진행하지 않은 구조에 관하여

    1. 계정 정보의 관하여
        - 보유 영웅 (현재 영웅 정보)
        - 보유 골드
        - 보유 아이템 (인벤토리)
        - 현재 영웅 장착중인 아이템
        - 현재 영웅 보유 특성
        - 세이브 및 로드 데이터

    2. 아이템의 관하여
        - 아이템 분류
        - 아이템 정보 (프로토 추가 아이템의 관하여 인덱스 형태)
        - 물리/마법 구분
        - 소비 아이템 사용
        - 아이템 파츠 구분
        - 장비 직업 제한
        - 전투 시 소비 아이템 갯수 제한
    
    3. 광장 시스템의 관하여
        - 영웅 소집(추가 영웅)
        - 영웅 훈련
        - 창고 확인 (인벤토리)
        - 숙소 (영웅 숫자 제한)
        - 던전 입장

    4. 보상 시스템의 관하여
        - 골드 획득
        - 아이템 획득
        - 영웅 추가 특성 획득
        - 영웅 추가 능력치 획득 (기본 능력치에 한하여 제공)

3. 프로토타입에 필요하지 않는 시스템 정리

    1. 가치관 시스템
    2. 파티 스텟은 일부 구현 (럭 시스템은 이번 프로토에서 제외)
    3. 스테이지 중간 결과

4. 후속 논의가 필요한 시스템
    
    1. 조건부 스텟의 경우 다른 기초 구조 완성 후 진행
    2. 파티 스텟 - 골드 추가 획득
